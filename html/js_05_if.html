javascript 조건문 이해하기

조건문의 정의:어떠한 조건이 부여되고, 참/거짓에 맞는 상황에 따라 선택 또는 행위를 취하는 것에 따른 문법

  • 삼항 연산자 : 조건문은 아니지만 두가지의 조건상황에 맞게 연산하게 되는 항
  • 단수 분기 : if/else if - 하나의 조건을 보고 그것이 참이면 수행 거짓이면 다른것을 하거나, 내용을 끝내는 기능으로 하나의 기능만을 수행하게 하는 형태
  • 다수 분기 : switch - 하나의 조건을 보고 그것이 참에 해당하는 경우, 수행할 수 있는 경우의 수를 모두 행하도록 하는 기능
    단, 경우의 상황이 끝나는 경우에는 반드시 break 구문을 작성해야함
